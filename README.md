# CMYW Studio: 物理级减色法光影画生成器

这是一个专为 Bambu Lab 打印机设计的 CMYW 多色光影画（Lithophane）生成工具。它能够将普通彩色图片转换为基于 CMYW 物理减色原理的 3D 模型 (.3mf)。

## ✨ 核心功能
* **1:1 物理渲染**：预览界面与打印实物保持 1:1 物理比例，确保切片效果心中有数。
* **专业级 CMYW 映射**：基于物理减色模型，利用四色耗材（青、品红、黄、白）实现高动态范围的光影还原。
* **拓竹生态友好**：直接导出 3MF 格式，自动识别分层对象，拖入 Bambu Studio 即可直接打印。
* **绿色无需安装**：打包为独立可执行程序，无需繁琐的 Python 环境配置。

## 🚀 如何使用
1. **启动软件**：双击 `CMYW_Studio.exe`。
2. **导入图片**：点击“选择图片”按钮导入你的照片。
3. **设置参数**：
    * 打印宽度：根据你的灯箱底座尺寸设定。
    * 分辨率：推荐 300 (适配 0.4 喷嘴) 或 500 (适配 0.2 喷嘴)。
4. **预览与检查**：点击渲染预览，确认比例无误。
5. **导出**：保存为 `.3mf` 文件，拖入 Bambu Studio。

## 🛠️ 技术栈
* **算法引擎**：OpenCV (图像处理) + Numpy (矩阵计算)
* **3D 引擎**：Open3D (网格编译与实时渲染)
* **GUI 框架**：Tkinter (原生轻量化界面)
* **文件标准**：3MF (3D Manufacturing Format)

## 💡 打印建议
* **耗材选择**：建议使用 Matte White PLA 作为底板，色彩层使用普通 PLA。
* **打印层高**：推荐 0.08mm 以获得最佳视觉效果。
* **喷嘴要求**：本程序针对 0.4mm 喷嘴优化，若使用 0.2mm 喷嘴可尝试 500 分辨率。
